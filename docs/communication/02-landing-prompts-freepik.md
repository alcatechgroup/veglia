# DICAS RÁPIDAS DE GERAÇÃO

**Para Freepik AI:**
- Selecione *Photorealistic* + *Editorial*
- Estilo: *Magazine* ou *Documentary*
- Cole o prompt exatamente como está

**Para Midjourney:**
- Adicione `--ar 1:1` (ou `--ar 4:3` / `--ar 16:9` / `--ar 4:5` conforme indicado)
- Acrescente `--style raw --v 6` para evitar over-smoothing
- Acrescente `--no text, logos, watermark`

**Para Adobe Firefly:**
- Estilo de conteúdo: *Photo*
- Configurações de cor: *Pastel* ou *Cool*
- Iluminação: *Natural light*

**Princípios não-negociáveis (revisão antes de aprovar):**

✓ Pessoas brasileiras, etnias autênticas, sorrisos genuínos
✓ Iluminação natural soft, sem dramaticidade
✓ Paleta dominante: azul-pastel + cream + acentos mint
✓ Aspect ratio correto para a posição na landing
✓ Sem texto, logos, números visíveis nas imagens
✓ Sem trajes formais com gravata (parece corporate frio)
✓ Sem stethoscopes ou ícones médicos genéricos
✓ Sem filtros saturados ou tons cinza-azul SaaS frio
